# 📊 Data Science & AI Projects – MBA Portfolio

This repository contains a collection of projects developed during my MBA in Data Science and Artificial Intelligence. The objective is to consolidate practical applications of concepts learned throughout the program, covering data analysis, machine learning, and problem-solving using real-world datasets.

## 🚀 Objectives

* Apply statistical and machine learning techniques to real datasets
* Develop end-to-end data analysis workflows
* Strengthen skills in data visualization and storytelling
* Explore practical use cases in Data Science and AI

## 🧠 Topics Covered

* Exploratory Data Analysis (EDA)
* Data Cleaning and Preprocessing
* Supervised and Unsupervised Learning
* Model Evaluation and Validation
* Data Visualization

## 🛠️ Tools & Technologies

* Python (Pandas, NumPy, Scikit-learn, Plotly, Matplotlib)
* Jupyter Notebook
* Git & GitHub

## 📁 Repository Structure

```
├── notebooks/        # Jupyter notebooks with analysis and experiments
├── data/             # Datasets (if applicable)
├── src/              # Supporting scripts and reusable code
├── README.md         # Project documentation

```

## 📌 Notes

Each project focuses on a specific problem or technique and is documented within its respective notebook. The emphasis is on clarity, reproducibility, and practical application of concepts.

# 📊 Projetos de Ciência de Dados e IA – Portfólio MBA

O objetivo é consolidar aplicações práticas dos conceitos aprendidos ao longo do programa, abrangendo análise de dados, machine learning e resolução de problemas com dados reais.

## 🚀 Objetivos

* Aplicar técnicas estatísticas e de machine learning em dados reais

* Aprendizado Supervisionado e Não Supervisionado

* Visualização de Dados

## 🛠️ Ferramentas e Tecnologias

* Python (Pandas, NumPy, Scikit-learn, Plotly, Matplotlib)
* Jupyter Notebook
* Git & GitHub

## 📁 Estrutura do Repositório

